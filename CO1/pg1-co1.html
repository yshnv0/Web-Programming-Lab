Core Subjects

  • Advanced Data Structure(ADS)

    • This subject deals with different types of data structures.
    It is concerned with various data structures such as Array,Queue,Stack,etc.

  • Advanced Software Engineering(ASE)

    • This subject deals with different software engineering mechanisms.
    It is concerned with various properties of software and life cycle of software.

  • Mathematical Founadtion for Computing(MFC)

    • Mathematical Founadtion for Computing deals with several maths-related topics.
    The topics include Sets,Integration,Differeneciation,etc.

  • Digital Fundamentals & Computer Architecture(DFC)

    • It deals with understanding fundamentals behind digital logic design.
    It helps to gain knowledge about digital circuits and systems.
